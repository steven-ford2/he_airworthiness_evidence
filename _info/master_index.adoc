== Design Philosophy

This repository enforces:

* Separation between theory-driven certification logic and mission-system fluency.
* Bidirectional coupling between research and compliance.
* Explicit context modeling (environment, theatre, mission type).
* Measurable human performance claims.
* Traceable requirement → hazard → design → evidence chains.
* Standardized artifact templates.
